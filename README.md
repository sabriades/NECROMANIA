# NECROMANIA

2D endless runner game project developed in C# using Unity during my time at the Apple Developer Academy in two weeks.   
NECROMANIA represents one of my most meaningful projects: fully designed, scripted and assembled by me.

---

## Tech Stack
- **Programming language:** C#
- **Game engine:** Unity
- **Tools:** Unity Editor, Visual Studio, Procreate

---

## Main Features
- Basic player movement and controls
- Obstacles
- Scene setup and asset integration

---

## Notes
NECROMANIA is a personal passion project and an early exploration of game development and software architecture.

*Project created for learning and portfolio purposes.*
